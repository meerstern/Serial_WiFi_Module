# Serial_WiFi_Module
 AMW037 module board

## 概要 
  * SiliconLabs製WiFiモジュールAMW037を搭載した変換基板です。  
  * WiFi制御ソフトウェアが予め書き込まれているため、容易にWiFi通信を使⽤することができます。
  * 3.3Vシリアル通信(UART,初期115200bps)で外部機器からパラメータを設定することが可能です。  
  * 技適、FCCマーキング有。
